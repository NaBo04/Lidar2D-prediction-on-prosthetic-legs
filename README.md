# Contenido de este Repositorio :books:

    - DocumentosContexto: Contiene los documento entregados por Canvas que indican las instrucciones
        de las entregas. Además se incluyen los Papers relacionados al proyecto.
    - Entregables: Incluye los informes, presentaciones o similar, desarrollados por el equipo para
        las entregas del curso.
    - Reuniones: Contiene lo relacionado a las reuniones con la contraparte del proyecto, como los
        apuntes.
